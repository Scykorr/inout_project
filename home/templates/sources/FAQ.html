Вы оказываете услуги по внедрению?

Да, услуга профессионального внедрения доступна для тарифных планов «Профессиональный», «Командный», «Бизнес» и «Корпоративный». Процессом внедрения руководят опытные, сертифицированные менеджеры. Знакомьтесь с нашей командой по внедрению. Мы будем рады вам помочь!

Какие методологии проектного менеджмента лежат в основе Easy Project?

В основе Easy Project лежит принцип тройственной ограниченности. Для управления содержанием применяется иерархическая структура работ (WBS), графики создаются в диаграмме Гантта, а генеральный план помогает распределять ресурсы. Ход выполнения проекта оценивается по кривым освоенного объёма, для анализа риска используется иерархическая структура рисков, а панели Kanban позволяют контролировать ежедневную работу. Кроме того, в Easy Project предусмотрены модули службы поддержки, управления финансами и CRM. В скором времени добавится возможность интеграции разработки и эксплуатации, а также интеллектуальный анализ данных.

Какие тарифные планы предлагает Easy Project?

  • База - включает в себя все классические инструменты управления проектами, такие как управление работой, гибкое управление, отслеживание времени, управление портфелем, контроль проекта и отчетность
  • Бизнес - включает в себя все основные инструменты PM плюс управление ресурсами и финансами
  • Платформа - включает все инструменты База и Бизнес PM, а также CRM и службу поддержки

Для каких сфер управления подходит Easy Project?

Управление проектами, портфелями проектов и программами, гибкие команды, стратегические проекты, разработка продуктов, инновации, большие и маленькие команды, корпоративные проекты, интеграция разработки и эксплуатации, ИТ-проекты, управление взаимоотношениями с заказчиками, управление финансами и не только.

В чём разница между консультированием и клиентской поддержкой?

Функции отдела консультирования/внедрения:

  • Платные услуги
  • Анализ и настройка приложения консультантом с учетом требований клиента; пересмотр и реконфигурация настроек для существующих клиентов по мере необходимости
  • Обучение администраторов и конечных пользователей
  • Интеграция/импорт данных/миграция данных: анализ, тестирование, внутренняя координация проекта, управление
  • Запрос на индивидуальную разработку: анализ, тестирование, внутренняя координация проекта разработки, управление

Функции отдела клиентской поддержки:

  • Первичное техническое обслуживание клиентов:
  • Решение базовых технических проблем (поддержка 1-го уровня)
  • Онлайн-чат: базовая техническая поддержка
  • Решение проблем, связанных с простоем приложения
  • Устранение неполадок в клиентской зоне и проблем с доступом
  • Устранение сообщений об ошибках (ошибка 500, 403 и так далее)
  • Ответы на вопросы из серии «где найти» или запросы, связанные с общими настройками приложения
  • Передача клиента в отдел консультирования для решения вопросов, связанных с пользовательской настройкой или сложными настройками, требующими больше времени и анализа (клиенты на этапе внедрения)
  • Передача клиента в отдел продаж для обработки коммерческих запросов или оплаты услуг по консультированию (клиенты на этапе после запуска приложения в работу)
  • Предоставление ссылки на доступную документацию с подробными инструкциями о работе приложения
  • Помощь клиентам, сообщившим об ошибке или отправившим запрос на индивидуальную разработку
  • Помощь в решении проблем с клиентским сервером (платная услуга, если требуется привлечение специалистов поддержки 2-го уровня)

Что подразумевается под «сервером» и «облаком»?

Серверное решение

Серверное решение разработано для вашего собственного сервера (то есть программное обеспечение размещено на ваших ресурсах). Покупая это решение, вы получаете комплект ПО для установки и исходный код.

Облачное решение

Облачное решение предусматривает хранение данных на удалённых серверах Easy Software Ltd. Для клиентов из США мы используем хранилище Amazon Cloud в Орегоне. При покупке данного решения вы получаете право использовать программное обеспечение в облаке в течение установленного срока. Облачное решение Easy Project включает:

  • EP 2019 + отдельные модули и расширения
  • Обновления ПО — актуальная версия Easy Project всегда в вашем распоряжении
  • Ежедневное резервирование, хранение 14 резервных копий
  • Защита по SSL
  • Гарантия восстановления после сбоя в течение 1 часа

Что подразумевается под «сервером» и «облаком»?

Центры хранения и обработки данных по всему миру

Защита данных

  • Хранение данных на серверах в простом формате с возможностью локального шифрования.
  • Данные хранятся только на сервере со строго ограниченным доступом
  • Соглашение о конфиденциальности и полное стирание данных администраторами сервера

План восстановления

В хранилище Easy Cloud предусмотрено несколько планов восстановления, которые регулярно тестируются и обновляются. Планы восстановления охватывают следующие проблемы:

  • Неисправность сервера
  • Сбой приложения
  • Сбой центра хранения и обработки данных
  • Утечка учётных данных

Резервирование

Резервные копии хранятся в разных точках мира для обеспечения безопасности в случае стихийного бедствия или серверного сбоя.

Резервные копии создаются ежедневно и хранятся не менее 3 месяцев.

  • Зашифрованная передача данных
  • Хранение данных в разных местах
  • Доступ к процессу резервирования и зарезервированным данным через сервер невозможен по соображениям безопасности
  • Утечка учётных данных
