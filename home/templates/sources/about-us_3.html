О нас

Easy Software был основан в 2006 году. Через год мы уже запустили наше собственное программное обеспечение для управления проектами. В 2012 году наши продукты по управлению проектами стали доступны во всем мире.

По сей день, более чем 3.500 компаний по всему миру преобрели наши продукты Easy Project и Easy Redmine. Наши продукты представяют собой высокоадаптируемые решения для управления проектами, которые используются во всех промышленных секторах с сильной позицией в технологиях и инженерии.

Easy Software используется не только для управления проектами. Easy Software - это культура, навеянная гармонией.
